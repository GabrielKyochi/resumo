Html

• HTML é um marcador de texto

• O HTML é feito em um arquivo index

• No HMTL é possível usar o comando Doc ou !Docytype, mas é preferível usar o !Docytype, pois ele é mais recente.

O Doc é o comando inicial do HTML e ele é usado para desenvolver a estrutura básica, ao usá-lo, ele fará automaticamente:
html lang="en"
head
meta charset="UTF-8"
meta name="viewport" content="width=device-width, initial-scale=1.0"
title Resumo para a Prova title
head
body
...
body

• Após realizar o doc, é possível adicionar dois dados novos, link e script, que são muito importantes. O link, é feito da seguinda forma: link rel="stylesheet" href="arquivoDeConexão", e geralmente é utilizado para ligar o HTML com o CSS. Já o script, é feito dessa maneira: script src="script.js" script, e é utilizado para ligar o HTML com o javascript ou script.js, como geralmente é nomeado.
